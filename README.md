## What are Kavela Skills?

[Kavela](https://kavela.ai) is a shared context platform for AI tools. A **skill** is a unit of knowledge -best practices, coding standards, runbooks, design systems -that AI assistants (Claude, Cursor, etc.) can reference automatically via the [Model Context Protocol (MCP)](https://modelcontextprotocol.io).

When teams publish skills on the Kavela marketplace as **free**, they appear here so anyone can:

- **Browse** the knowledge directly on GitHub
- **Fork** and adapt skills for their own team
- **Install** them into their Kavela workspace with one click from the [marketplace](https://kavela.ai/marketplace)
- **Reference** them in any MCP-compatible AI tool

## How It Works

This repo is **automatically maintained** by the Kavela platform:

- When a user **publishes** a free skill on the marketplace, it's pushed here within seconds
- When a listing is **updated**, the files are updated with a new commit
- When a listing is **removed or delisted**, the files are deleted
- Every commit is **co-authored** by the original publisher

You never need to manually edit this repo -it's a living mirror of the marketplace.

## Security Notice

> **Use at your own risk.** Skills in this repository are community-contributed and may not have been reviewed or scanned for malicious content. Always review skill content before using it in production environments. Kavela does not guarantee the safety, accuracy, or quality of community-published skills.
>
> If you find a skill with harmful or inappropriate content, please [report it](https://kavela.ai/report) or open an issue on this repo.
